Function Symmetry

Odd Functions \(f(-x) = -f(x)\)

  • Defined by (both are the same)
    \(f(-x) = -f(x)\)
    \(f(x) = -f(-x)\)

  • Odd functions are symmetric with respect to the origin.

    The positive and negative sides have equal magnitude; going in opposite directions.

Example graph of \(y = x^3\)

The following is a graph of \(y = x^3\), notice how the left and right sides have equal magnitude, but in opposite directions.

Be aware of transformations to the original function of \(f(x) = x^3\)

(Note that these functions are not odd.)

Functions that are neither even nor odd

Example

The following is a graph of \(f(x) = x^2 - 2x\).

Since

  • \(f(-1) = 1 - 1 = 0\)
  • \(f(1) = 1 + 1 = 2\)

Therefore

Not even
\(f(-x) \neq f(x)\)
Not odd
\(f(-x) \neq -f(x)\)
Therefore
This function is neither even nor odd.
