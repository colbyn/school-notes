Approximate Area

$$\begin{equation} \begin{split} R_n &= f(x_1)\Delta x + f(x_2)\Delta x + \cdots + f(x_n)\Delta x &= \sum_{i = 1}^{n} f(x_i)\Delta x \;\; (1)\;(2) \\ \end{split} \end{equation}$$

Definition 2

The area A of the region S that lies under the graph of the continuous function \(f\) is the limit of the sum of the areas of approximating rectangles:

$$\begin{equation} \begin{split} A &= \lim_{n \to \infty} R_n = \lim_{n \to \infty} \left[ f(x_1)\Delta x + f(x_2)\Delta x + \cdots + f(x_n)\Delta x \right] \;\; (2) \\ \\ A &= \lim_{n \to \infty} L_n = \lim_{n \to \infty} \left[ f(x_0)\Delta x + f(x_1)\Delta x + \cdots + f(x_{n - 1})\Delta x \right] \;\; (3) \end{split} \end{equation}$$

Using sigma notation

$$\begin{equation} \begin{split} A &= \lim_{n\to\infty} \sum_{i = 1}^n f(x_i) \Delta x \\ A &= \lim_{n\to\infty} \sum_{i = 1}^n f(x_{i - 1}) \Delta x \\ A &= \lim_{n\to\infty} \sum_{i = 1}^n f(x_i^*) \Delta x \\ \end{split} \end{equation}$$

In fact, instead of using left endpoints or right endpoints, we could take the height of the \(i\text{th}\) rectangle to be the value of \(f\) at any number \(x_{i}^{*}\) in the \(i\text{th}\) subinterval \([x_{i - 1}, x_i]\).

We call the numbers \(x_{1}^{*}, x_{2}^{*}, \cdots , x_{n}^{*}\) the sample points.

The following shows approximating rectangles when the sample points are not chosen to be endpoints. So a more general expression for the area of \(S\) is:

$$A = \lim_{x \to \infty} \left[ f(x_{1}^{*})\Delta x + f(x_{2}^{*})\Delta x + \cdots + f(x_{n}^{*})\Delta x \right]$$

Left Riemann Sum
\(x_{i}^{*} = x_{i - 1} \;\; \text{for all $i$}\)
Right Riemann Sum
\(x_{i}^{*} = x_{i} \;\; \text{for all $i$}\)
Middle Riemann Sum
\(x_{i}^{*} = \frac{1}{2}(x_{i} + x_{i + 1}) \;\; \text{for all $i$}\)

\(x_{i}^{*}\) may refer to right endpoints explicitly.

Riemann sums

Left Riemann sum
$$\sum_{i = 0}^{n - 1} \Delta x \cdot f(x_i)$$
Right Riemann sum
$$\sum_{i = 1}^{n} \Delta x \cdot f(x_i)$$

The Distance Problem

The Distance Problem
find the distance traveled by an object during a certain time period if the velocity of the object is known at all times.

If the velocity remains constant, then the distance problem is easy to solve by means of the formula:

$$\begin{equation} \begin{split} \text{distance} &= \text{velocity} \cdot \text{time} \\ \text{distance} &= \text{rate} \cdot \text{time} \end{split} \end{equation}$$

Integral Formulas

Properties of the Definite Integral:

$$\begin{equation} \begin{split} \I{f(x)} &= - \Integrate{b}{a}{f(x)} \\\\ \Integrate{b}{a}{f(x)} &= - \I{f(x)} \\\\ \Integrate{a}{a}{f(x)} &= 0 \\\\ \I{c} &= c(b - a) \;\; \text{if $c$ is any constant}\; \\\\ \I{\left[ f(x) \pm g(x) \right]} &= \I{f(x)} \pm \I{g(x)}\\\\ \I{c \cdot f(x)} &= c \cdot \I{ f(x)} \\\\ \Integrate{a}{c}{f(x)} &= \Integrate{a}{b}{f(x)} + \Integrate{b}{c}{f(x)} \end{split} \end{equation}$$

Comparison Properties of Integrals (Properties 6 to 8)

  • If \(f(x) \geq 0\) for all \(a \leq x \leq b\), then \(\I{f(x)} \geq 0\)

  • If \(f(x) \geq g(x)\) for all \(a \leq x \leq b\), then \(\I{f(x)} \geq \I{g(x)}\)

  • If \(m \leq f(x) \leq M\) for all \(a \leq x \leq b\), then \(m(b - a) \leq \I{f(x)} \leq M(b - a)\)

4.3 | The Fundamental Theorem of Calculus

the Fundamental theorem of calculus, Part 2

If \(f\) is continuous on \([a, b]\), then:

$$\I{f(x)} = F(b) - F(a)$$

Where \(F^\prime = f\).

Or Alternatively

$$\I{F^\prime(x)} = F(b) - F(a)$$

the Fundamental theorem of calculus

  1. If \(g(x) = \IntegrateFor{a}{x}{f(t)}{t}\), then \(g^\prime(x) = f(x)\)
  2. \(\I{f(x)} = F(b) - F(a)\), where \(F^\prime = f\).

Part 1 can be rewritten as

$$\newcommand{\derivative}{\frac{\mathrm{d}}{\mathrm{d}x}\;} \begin{equation} \begin{split} \derivative \IntegrateFor{a}{x}{f(t)}{t} &= f(x) \end{split} \end{equation}$$

When applying Part 1, pay attention to the upper and lower bounds.

4.4 | Indefinite Integrals and the Net Change Theorem

You should distinguish carefully between definite and indefinite integrals. A definite integral \(\I{f(x)}\) is a number, whereas an indefinite integral \(\int f(x) dx\) is a function (or family of functions).

Integral Formulas

$$\newcommand{\In}[1]{\int {#1} \; \mathrm{d}x} \begin{equation} \begin{split} \In{c f(x)} &= c \In{f(x)} + C \\ \In{k} &= k x + C \\ \In{f(x) + g(x)} &= \int f(x) + \int g(x) + C \\ \In{x^n} \; \text{if}\; x \neq -1 &= \frac{x^{n + 1}}{n + 1} + C \\ \In{\cos(x)} &= \sin(x) + C \\ \In{\sin(x)} &= -\cos(x) + C \\ \In{\sec^2(x)} &= \tan(x) + C \\ \In{\tan(x) \cdot \sec(x)} &= \sec(x) + C \\ \In{\csc^2(x)} &= -\cot(x) + C \\ \In{\cot(x)\csc(x)} &= -\csc(x) + C \end{split} \end{equation}$$

Definition 2 - Total Displacement

Given:

  • A position function \(s(t)\)
  • The derivative of \(s(t)\), i.e. the velocity function \(v(t)\)
$$\IntegrateFor{t_1}{t_2}{v(t)}{t} = s(t_2) - s(t_1)$$

is the net change of position, or displacement, of the particle during the time period from \(t_1\) to \(t_2\).

This may only work for an object moving in a straight line.

Definition 3 - Total Distance

Given:

  • A position function \(s(t)\)
  • The derivative of \(s(t)\), i.e. the velocity function \(v(t)\)

We can compute the distance by integrating the magnitude of the velocity function:

$$\IntegrateFor{t_1}{t_2}{|v(t)|}{t} = \text{total distance traveled}$$

4.5 | The Substitution Rule

Theorem 4 - The Substitution Rule

If \(u = g(x)\) is a differentiable function whose range is an interval \(I\) and \(f\) is continuous on \(I\), then:

$$\newcommand{\In}[1]{\int {#1} \; \mathrm{d}x} \newcommand{\InU}[1]{\int {#1} \; \mathrm{d}u} \In{(f \circ g)(x) \; g^\prime(x)} = \InU{f(u)}$$

U Substitution Alternative

Consider the result as a product of \(A\) and \(B\). Identify \(A\) as the original function applied to the chain rule, and rewrite it in terms of a derivative to make it clearer, then use the following framework to find \(B\).

$$\begin{align*} \int\frac{\cos(5x)}{e^{sin(5x)}} \mathrm{d}x &= AB + C \;\; \text{where}\left\{ \begin{array}{ll} A = \int \frac{\mathrm{d}}{\mathrm{d}x} e^{-\sin(5x)} \end{array} \right. \end{align*}$$

If \(A = e^{-\sin(5x)}\), we need to eliminate what is left (the 'Leftovers' above), and find \(B\). Setup the equation using the following framework:

  • \(\text{Leftovers} = \cos(5x)\)
  • \(\text{Artifacts} = \text{Artifacts from $\frac{\mathrm{d}}{\mathrm{d}x} A$}\)
$$\begin{align*} \text{Leftovers} &= B \cdot \left(\text{Artifacts from $\frac{\mathrm{d}}{\mathrm{d}x} f(x)$}\right) \end{align*}$$
$$\begin{align*} \overbrace{\cos(5x)}^{\text{Leftovers}} = B \cdot \overbrace{\frac{\mathrm{d}}{\mathrm{d}x} \left( -\sin(5x) \right)}^{\text{Artifacts}} &= - A \cdot 5\cos(5x) \\ -\frac{\cos(5x)}{5\cos(5x)} & = A = -\frac{1}{5} \end{align*}$$
$$\begin{equation} \int\frac{\cos(5x)}{e^{sin(5x)}} \mathrm{d}x = -\frac{1}{5}e^{-\sin(5x)} + C \end{equation}$$

I reduced U-Substitution to a simple formula. :)

Theorem 5 - The Substitution Rule for Definite Integrals

If \(g^\prime\) is continuous on \([a, b]\) and \(f\) is continuous on the range of \(u = g(x)\), then:

$$\newcommand{\In}[1]{\int {#1} \; \mathrm{d}x} \newcommand{\InU}[1]{\int {#1} \; \mathrm{d}u} \I{(f \circ g)(x) \; g^\prime(x)} = \IntegrateFor{g(a)}{g(b)}{f(u)}{u}$$
