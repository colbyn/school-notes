Definition of stem:[\arccos(x)]

y = \arccos(x) if and only if x = \cos(y) where -1 \leq x \leq 1 \\ 0 \leq y \leq \pi The graph of stem:[y=\cos\;x] with stem:[x] restricted to to stem:[\left\lbrack 0,\frac{1}{2}\tau \right\rbrack]: The choice of ranges for stem:[\arcsin(x)] and stem:[\arccos(x)] is not universally accepted.
