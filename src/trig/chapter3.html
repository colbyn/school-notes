§3.5 | Inverse Trigonometric Functions

Definition of stem:[\arcsin(x)]

y = \arcsin(x) if and only if x = \sin(y) where -1 \leq x \leq 1 \\ -\frac{1}{4}\tau \leq y \leq \frac{1}{4}\tau The graph of stem:[y=\sin\;x] with stem:[x] restricted to to stem:[\left\lbrack -\frac{\tau}{4},\frac{\tau}{4} \right\rbrack]:

Definition of stem:[\arccos(x)]

y = \arccos(x) if and only if x = \cos(y) where -1 \leq x \leq 1 \\ 0 \leq y \leq \pi The graph of stem:[y=\cos\;x] with stem:[x] restricted to to stem:[\left\lbrack 0,\frac{1}{2}\tau \right\rbrack]: The choice of ranges for stem:[\arcsin(x)] and stem:[\arccos(x)] is not universally accepted.

Inverse Reciprocal Identities

\begin{equation} \begin{split} \csc^{-1}(x) &= \sin^{-1} \left( \frac{1}{x} \right) &\;\;\; \text{if}\; & -1 \leq x \leq 1 \\ &\\ \sec^{-1}(x) &= \cos^{-1} \left( \frac{1}{x} \right) &\;\;\; \text{if}\; & -1 \leq x \leq 1 \\ &\\ \cot^{-1}(x) &= \frac{\pi}{2} - \tan^{-1}(x) &\;\;\; \text{if}\; & x \in \mathbf{R} \\ \end{split} \end{equation}

Other Inverse Co-Tangent Identities

\begin{equation} \begin{split} \cot^{-1}(x) = \tan^{-1}\left(\frac{1}{x}\right) &\;\;\; \text{if}\; & x \gt 0 \end{split} \end{equation}

§3.6 | Trigonometric Equations

It's just like solving for quadratic equations:

\begin{equation} \begin{split} 4\sin x \cdot \cos x - 2\sqrt{3}\sin x - 2\sqrt{2}\cos x + \sqrt{6} &= 0 \\ 2\sin x \left( 2\cos x - \sqrt{3} \right) - \sqrt{2} \left( 2\cos x - \sqrt{3} \right) &= 0 \\ \left(2\sin x-\sqrt{2} \right) \left(2\cos x - \sqrt{3}\right) &= 0 \\ &\\ 2\sin x - \sqrt{2} &= 0 \\ 2\cos x - \sqrt{3} &= 0 \\ &\\ x &= \frac{3}{4}\pi, \frac{1}{4}\pi, \frac{1}{6}\pi, \frac{11}{6}\pi \end{split} \end{equation}
